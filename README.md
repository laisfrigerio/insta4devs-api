# insta4devs-api

AWS Lambda service to create posts, save it into AWS DynamoDb and upload images to S3. Usage of SQS queue to dealing with async messages.

## Stack

- Node.js 22
- TypeScript 4
- Jest
- AWS Cloud (Lambda, S3, DynamoDb, SQS)
